# Oculink eGPU 热插拔助手 (Windows)

这是一个专门为使用 **Oculink** 连接外接显卡 (eGPU) 的用户设计的管理脚本。由于 Oculink 本质上是直连 PCIe 通道，不支持像 USB4/雷电那样的“原生热插拔”控制器，直接物理断开连接极易导致系统蓝屏 (BSOD)、数据丢失或驱动损坏。

本工具通过 **PowerShell** 调用 Windows 系统底层 PnP (即插即用) 指令，实现软件层面的“安全弹出”与“快速挂载”。



## 🛠️ 使用前准备

### 文件准备
* 下载 `OculinkManager.ps1` (脚本) 或 `OculinkManager.exe` (封装版) 。

---

## 📖 操作流程 (必读)

### 🔗 情况 A：【首次连接】或【物理重新插拔】
1.  **物理连接**：插好 Oculink 线并开启 eGPU 电源。
2.  **执行扫描**：运行工具，输入 **`1`** (扫描硬件改动)。
3.  **检查状态**：如果显卡显示为 `OK`，即可直接使用。如果显示为 `Disabled`，请执行下方“情况 B”的步骤。

### 🔄 情况 B：【设备已连接但处于禁用状态】
> **注意**：如果你上次使用时点击了“禁用”，那么即使执行“功能 1 扫描”也无法直接使用，必须手动解除禁用。
1.  **执行开关**：运行工具，输入 **`2`** (启用/禁用外接显卡)。
2.  **手动启用**：在列表中找到状态不是 `OK` 的外接显卡，输入其序号，确认**“启用”**。
3.  **生效**：状态变为 `OK` 后，显卡正式恢复工作。

### 🔌 情况 C：【准备拔出】eGPU
1.  **清理进程**：关闭所有正在使用显卡的程序（游戏、浏览器等）。
2.  **执行禁用**：运行工具，输入 **`2`**，选择外接显卡并确认**“禁用”**。
3.  **物理断开**：待脚本提示禁用成功，先关电源，再拔线。


## ❓ 常见问题 FAQ

**Q: 为什么生成的 .exe 文件会报毒？**
A: 这是因为 PS2EXE 封装工具产生的程序没有数字签名，且行为涉及硬件底层控制，容易被杀毒软件误判。请放心添加信任或直接使用 `.ps1` 脚本。

**Q: 扫描后找不到我的显卡？**
A: 检查 Oculink 线缆是否插紧，以及 eGPU 的电源开关是否已打开。部分笔记本可能需要重新插拔 Oculink 线缆并再次执行扫描。
