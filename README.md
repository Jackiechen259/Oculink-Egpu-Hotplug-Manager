# 🚀 Oculink eGPU 热插拔助手 (Windows)

这是一个专门为使用 **Oculink** 连接外接显卡 (eGPU) 的用户设计的管理脚本。由于 Oculink 本质上是直连 PCIe 通道，不支持像 USB4/雷电那样的“原生热插拔”控制器，直接物理断开连接极易导致系统蓝屏 (BSOD)、数据丢失或驱动损坏。

本工具通过 **PowerShell** 调用 Windows 系统底层 PnP (即插即用) 指令，实现软件层面的“安全弹出”与“快速挂载”。

---

## ✨ 核心功能
* **一键扫描硬件**：物理连接并通电后，快速触发系统检测新硬件，无需重启。
* **安全禁用/启用**：在物理拔出前，先通过软件指令让显卡“下线”，防止系统内核崩溃。
* **管理员权限自动提升**：内置权限校验，双击运行即可自动请求管理员身份。
* **状态实时反馈**：直观列出所有显卡及其当前工作状态。

---

## 🛠️ 使用前准备

### 1. BIOS 设置建议
为了提高稳定性，建议在主板 BIOS 中：
* 开启 **PCIe Hotplug**（如果主板支持）。
* 将 Oculink 对应的 PCIe 插槽链路速度固定为 **Gen3** 或 **Gen4**（避免 Auto 模式下握手失败）。

### 2. 文件准备
* 确保 `OculinkManager.ps1` (脚本) 或 `OculinkManager.exe` (封装版) 放在容易找的地方。

---

## 📖 操作流程 (必读)

### 🔗 当你需要【连接】eGPU 时：
1.  **物理连接**：插好 Oculink 线缆。
2.  **开启电源**：开启外接显卡的供电电源。
3.  **运行脚本**：打开本工具，输入 **`1`** (扫描硬件改动)。
4.  **确认**：听到系统“嘀嗒”一声后，显卡将在设备管理器中上线。

### 🔌 当你需要【拔出】eGPU 时：
1.  **清理进程**：**必须**先关闭所有正在使用外接显卡的程序（如游戏、视频剪辑软件、使用硬件加速的浏览器等）。
2.  **软件禁用**：运行本工具，输入 **`2`** (启用/禁用外接显卡)。
3.  **选择设备**：在列表中输入外接显卡对应的序号，确认将其**禁用**。
4.  **物理断开**：待脚本提示“安全移除”后，先关掉外接电源，再拔掉 Oculink 线缆。

---

## ⚠️ 极其重要 (危险预防)
> **死机/蓝屏预防准则**
> * **严禁**在显卡还在高负载运行或程序占用时点击“禁用”。
> * **严禁**在未执行软件“禁用”操作的情况下直接拔掉 Oculink 线缆。这在大多数 Windows 系统上会直接导致死机或自动重启。

## ❓ 常见问题 FAQ

**Q: 为什么生成的 .exe 文件会报毒？**
A: 这是因为 PS2EXE 封装工具产生的程序没有数字签名，且行为涉及硬件底层控制，容易被杀毒软件误判。请放心添加信任或直接使用 `.ps1` 脚本。

**Q: 扫描后找不到我的显卡？**
A: 检查 Oculink 线缆是否插紧，以及 eGPU 的电源开关是否已打开。部分笔记本可能需要重新插拔 Oculink 线缆并再次执行扫描。
